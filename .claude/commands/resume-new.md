# 恢复最新对话 (Resume Latest Conversation)

自动载入当前项目最新的 Claude 对话，快速恢复上次工作情景。

---

## 执行流程

### 第一步：定位项目对话文件夹

1. 获取当前工作目录的绝对路径
2. 将路径转换为 `.claude/projects/` 下的文件夹名称（`C:\Users\X1` → `C--Users-X1`）
3. 检查该项目文件夹是否存在

### 第二步：查找最新对话

1. 列出项目文件夹中所有 `.jsonl` 文件
2. 按修改时间降序排列
3. 选取最新的对话文件

### 第三步：解析对话信息

读取对话文件的第一行，提取以下信息：
- `summary`：对话摘要
- 从后续行提取：
  - `sessionId`：会话 ID
  - `timestamp`：最后更新时间
  - `cwd`：当时的工作目录

### 第四步：展示对话信息并交互确认

格式化展示找到的对话，使用 AskUserQuestion 工具提供选项：
1. ✅ 恢复并继续
2. 📋 查看更多对话
3. ❌ 取消

---

## 恢复对话后的行为

当用户选择"恢复并继续"时：
1. **阅读对话摘要**，理解上次工作的核心目标
2. **简短介绍**当时的工作情景（基于摘要）
3. **主动询问**：上次我们在做什么，现在想继续吗？

---

**现在执行**：检测当前项目的最新对话并准备恢复。
